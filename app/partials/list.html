Задание для кандидата на должность
JavaScript-developer

От Вас требуется сделать аналогичный сайт - одностраничное веб-приложение на KnockoutJS или AngularJS.

Сайт должен иметь три раздела: /Info, /Orientation, /Bezier. По умолчанию должна открываться раздел Info.
Сайт должен быть одностраничным, т.е. html-контент должен подгружаться динамически через Ajax.

  1. Info

    Просто копия этого раздела.

  2. Orientation

    Дан массив точек на плоскости, образующих замкнутую самонепересекающуюся ломаную линию - многоугольник. Определение ориентации многоугольника - разделение всех многоугольников на два класса - положительноориентированнные ("по часовой") и отрицательноориентированные ("против часовой").

    В /Data/Orientation.json находится массив массивов точек. Для каждого из них требуется определить ориентацию и отобразить результат на странице.

  3. Bezier

    По заданным точкам построить гладкую кривую Безье. Кривая должна проходить через все точки и не иметь изломов.

    В /Data/Bezier.json находится массив массивов точек. Для каждого из них требуется построить кривую Безье и отобразить на странице в виде svg-элемента path.
